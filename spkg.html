Shonky Package Manager

spkg is a small package management system I have written for Revol. It works as a "cross compiled" package system. Whereas most package managers run on their target system, this builds the root filesystem automatically on a host system. It is written in perl and is in some ways inspired by Debian's dpkg.

This system is the base for Revol 2.0.
